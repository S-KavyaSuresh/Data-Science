# Data Cleaning & Visualization

## 📌 Overview

Performed data cleaning and exploratory data analysis on a sleep dataset to understand patterns and trends.

## 📂 Dataset

* Original Dataset: [sleep.csv](Data_Cleaning_and_Visualization/dataset/sleep.csv)
* Cleaned Dataset: [cleaned_sleep_data.csv](Data_Cleaning_and_Visualization/cleaned_sleep_data.csv)

## 🔧 What I Did

* Loaded and explored the dataset using pandas
* Cleaned data (handled missing values & duplicates)
* Performed basic statistical analysis
* Created visualizations using matplotlib & seaborn

## 📊 Key Insights

* Observed distribution of sleep duration
* Identified common sleep disorders
* Noticed patterns between lifestyle factors and sleep

## 📚 What I Learned

* Importance of data cleaning before analysis
* How to visualize data effectively
* Basics of exploratory data analysis (EDA)
* Using pandas for real-world datasets
